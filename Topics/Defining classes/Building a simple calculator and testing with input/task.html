Building a simple calculator and testing with input

Hey there! This problem might be a bit unpredictable, but give it a go and let us know how you do!
Create a class 'Calculator' in Java, which has two private variables x and y of type integer. The class should have non-argument constructor initializing x and y to 0, and it should provide public set methods for x and y. It should have a method 'add' that returns the sum of x and y. Write a main method where you set values of x and y from the standard input and print the value returned by the 'add' method. The input will be two single spaced integers, the values of x and y respectively, and the output should be the sum of these integers.
Sample Input:
1 2

Sample Output:
3


Sample Input:
10 20

Sample Output:
30



Memory limit: 256 MB
Time limit: 5 seconds
